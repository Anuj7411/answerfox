# Answerable

> The drop-in SEO toolkit that makes any site answerable by AI search engines.

Install a few packages, run one command, and your Next.js site is SEO-ready, AI-search-ready, and audited continuously — without you becoming an SEO expert.

```bash
# In an existing Next.js project:
pnpm add @answerable-kit/schemas @answerable-kit/metadata @answerable-kit/sitemap @answerable-kit/audit

# Scaffold trust pages + sitemap + robots:
pnpm dlx @answerable-kit/cli init

# Audit the result against 33 production-tested checks:
pnpm dlx @answerable-kit/cli audit http://localhost:3000
```

A score of 85+ on first audit is the design target. The [`examples/basic-nextjs`](https://github.com/Anuj7411/answerable/tree/main/examples/basic-nextjs) example scores 90+ out of the box.

<Callout type="info">
  **v0.1.0 is live on npm** under the [`@answerable-kit/*`](https://www.npmjs.com/org/answerable-kit) scope. Currently shipping 33 of 50 audit checks — the remaining 17 land incrementally before v1.0.
</Callout>

## What's in the toolkit

| Package | Purpose |
| --- | --- |
| `@answerable-kit/core` | Branded URL types, errors, the `Check<T>` interface |
| `@answerable-kit/schemas` | Type-safe JSON-LD generators (8 of them) |
| `@answerable-kit/metadata` | `defineSeo()` — Next.js metadata helpers |
| `@answerable-kit/sitemap` | `buildSitemap()` with smart path-based defaults |
| `@answerable-kit/templates` | Five trust-signal page templates |
| `@answerable-kit/audit` | Cheerio-backed audit engine (33 of 50 checks) |
| `@answerable-kit/cli` | The `answerable` command |

## Why Answerable?

Indie developers and SaaS founders waste **20–40 hours per launch** on SEO that should be automated. No other tool combines drop-in code + automated audit + plain-English explanations in one MIT-licensed package — that's the wedge.

| | Drop-in fixes | Continuous audit | Plain-English teaching |
| :--- | :---: | :---: | :---: |
| `next-seo` | partial | ❌ | ❌ |
| Lighthouse | ❌ | partial | ❌ |
| Ahrefs / Semrush | ❌ | ✅ | ❌ |
| AI-SEO checklists | ❌ | partial | ✅ |
| **Answerable** | ✅ | ✅ | ✅ |
